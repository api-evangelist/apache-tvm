# Apache TVM (apache-tvm)
Apache TVM is an open-source compiler framework for deep learning that aims to provide performance portability across diverse hardware backends. It automates the optimization of deep learning models for deployment on CPUs, GPUs, and specialized accelerators.

## APIs

### Apache TVM
TVM provides a Python API for deep learning model compilation and optimization, Relay IR API for graph-level optimization, and runtime deployment APIs for deploying optimized models on CPUs, GPUs, and edge devices.
